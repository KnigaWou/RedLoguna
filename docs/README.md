# УВБ-76 Дашборд

Этот проект собирает, анализирует и визуализирует радиосообщения станции «The Buzzer».

## Что внутри
- Карта расшифрованных маршрутов (со сдвигом координат)
- Словарь кодов (`0010` → приказ, `0104` → смена частоты)
- Прогноз активности на основе исторических данных

## Как использовать
1. Откройте `index.html` в браузере.
2. Смотрите карту и таблицу.
3. В четверг в 13:00 по МСК запускайте запись эфира на 4625 кГц.

## Источники
- Логи за 2023–2026 из Telegram-каналов
- Авторский анализ паттернов и сдвигов координат
